# COVID-19 Data Analysis & Forecasting

## Project Overview
This project analyzes and forecasts COVID-19 infection trends using time-series modeling and data visualization.  
The goal is to understand regional infection waves and predict future case counts using machine learning.

##  Objectives
- Forecast regional COVID-19 infection rates
- Identify trends, waves, and seasonal patterns
- Compare infection dynamics across regions

##  Regions
Initial analysis focuses on Texas, United States of America. 

##  Tools & Technologies
- Python
- Pandas & NumPy
- Seaborn & Matplotlib
- TensorFlow / Keras
- Jupyter Notebooks
